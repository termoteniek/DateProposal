# Date Proposal for Eline

A simple, romantic one-page site to ask Eline out for pasta at an Italian restaurant. Her answer is emailed to you when she submits the form.

## Quick start

1. Open `index.html` in a browser to preview locally, or host the folder on any static host (GitHub Pages, Netlify, etc.).

2. **Set up email (required, free):**
   - Go to [https://web3forms.com](https://web3forms.com)
   - Enter **your** email address and create an access key
   - In `index.html`, replace `YOUR_ACCESS_KEY` in the hidden `access_key` field with your key

3. Share the live URL with Eline.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Page content and form |
| `styles.css` | Italian / warm styling with a subtle horse motif |
| `script.js` | Sends the form via Web3Forms and shows success/error messages |

## Customization

- Edit the copy in `index.html` (restaurant name, date ideas, etc.)
- Adjust colors and fonts in `styles.css` (`:root` variables at the top)
